Шаблон
	HREF:{ссылка} NUMBSTR:{кол-во страниц} PATHOUT:{путь к файлу вывода}
Образец
	HREF:https://habr.com/company/pvs-studio/ NUMBSTR:40 PATHOUT:C:\Users\Admin\Desktop\prjt\Output.csv
Одна строка - один блог. 
Глубину поиска можно задать больше максимального значения (в этом случае программа соберет все статьи, 
которые имеются в блоге).
Между тегами должен быть строго один пробел.
